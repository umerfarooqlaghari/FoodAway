# TakeAway Platform

TakeAway is a platform designed to reduce food waste by connecting customers with businesses that have surplus food at the end of the day.

## Structure

This repository is a monorepo containing three main components:
1. `backend/`: FastAPI backend server.
2. `admin-web/`: React (Vite) frontend for superadmin operations.
3. `mobile-app/`: React Native (Expo) mobile application.

## Quick Start

See the individual `README.md` files in each directory for specific setup instructions.
